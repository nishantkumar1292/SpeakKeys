<p>Sayboard — голосовая клавиатура (IME) с открытым исходным кодом для Android, использующая облачное распознавание речи. Поддерживаются два сервиса:</p>
<ul>
  <li><strong>Sarvam AI</strong> — оптимизирован для распознавания хинглиша (хинди + английский)</li>
  <li><strong>OpenAI Whisper</strong> — многоязычное распознавание речи</li>
</ul>
<br/>
<p>API-ключи настраиваются в Настройки > вкладка API. Доступны только сервисы с настроенным ключом.</p>
<br/>
<p>Sayboard использует следующие разрешения:</p>
<ul>
  <li>INTERNET — необходимо для облачного распознавания речи. Аудио отправляется в Sarvam AI и/или OpenAI для транскрипции.</li>
  <li>RECORD_AUDIO — ну, это ведь голосовая клавиатура.</li>
</ul>
